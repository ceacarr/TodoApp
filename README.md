React Todo List Uygulaması (TodoApp)
Bu proje, React ve Vite kullanılarak geliştirilmiş, kullanıcıların günlük görevlerini ekleyebileceği, işaretleyebileceği ve silebileceği
basit ve şık bir yapılacaklar listesi uygulamasıdır. Temel React prensiplerini ve modern arayüz tasarımını birleştiren bir uygulamadır.

✨ Özellikler
Görev Ekleme: Yeni görevleri listeye kolayca ekleme.
Görev Tamamlama: Tamamlanan görevleri işaretleme ve üzerini çizme.
Görev Silme: Listeden istenmeyen görevleri kalıcı olarak silme.
Modern Arayüz: Tailwind CSS ile hızlı, temiz ve tamamen responsive tasarıma sahiptir.

Kullanılan Teknolojiler
React: Kullanıcı arayüzü geliştirmek için temel kütüphane.
Vite: Hızlı geliştirme ortamı ve build aracı.
JavaScript (ES6+): Uygulamanın tüm iş mantığı.
Tailwind CSS: Hızlı ve esnek arayüz tasarımı için kullanılan utility-first CSS framework'ü.
